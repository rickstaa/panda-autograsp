Panda\+\_\+autograsp is an autonomous R\+OS based grasping solution that works with the \href{https://www.franka.de/panda/}{\texttt{ Panda Emika Franka robot}}. In this grasping solution, several opensource grasping solutions are implemented on the \href{https://www.franka.de/panda/}{\texttt{ Panda Emika Franka robots}} robot. These solutions work both on a physical as well as a simulated version of the panda robot. A simulated version of the panda robot is already shipped with this package.
